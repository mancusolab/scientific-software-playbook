# Scientific Agent Tools

`scientific-agent-tools` is an optional maintainer-facing plugin. It is for
people authoring or maintaining playbook plugins, skills, agents, directive
files, and repository context files. It is not required for normal downstream
scientific workflow execution.

This plugin includes material adapted from
`ed3d-extending-claude`; see [../NOTICE.ed3d](../NOTICE.ed3d),
[../LICENSE.ed3d](../LICENSE.ed3d), and [../LICENSE](../LICENSE).

## Skills

- `creating-a-plugin`: create or refit installable playbook plugins and wire them into repo contracts
- `creating-an-agent`: define focused delegate agents with clear triggers, tool bounds, and startup rules
- `maintaining-project-context`: keep `AGENTS.md` or `CLAUDE.md` aligned with contract changes
- `testing-skills-with-subagents`: validate that skills change behavior under baseline and pressure scenarios
- `writing-claude-directives`: write concise, behavioral instructions for skills, agents, and context files
- `writing-claude-md-files`: author `AGENTS.md` or `CLAUDE.md` files with correct scope and freshness rules
- `writing-skills`: structure, tighten, and validate `SKILL.md` files

## Agent

- `project-claude-librarian`: reviews implementation changes and coordinates context-file maintenance
